📊 Google Sheets Setup

1 Create Google Sheet: Open Google Sheets and create a new spreadsheet for your RM numbers.
2 Open Apps Script: In your sheet, go to Extensions → Apps Script.
3 Copy Script Code: Replace the default code with the provided Google Apps Script code.
4 Deploy Web App: Click Deploy → New deployment → Choose "Web app".
5 Set Permissions: Execute as "Me", Who has access "Anyone".
6 Copy URL: After deployment, copy the Web app URL and paste it above.
✅ Once configured: RM numbers will be automatically logged with date and time stamps when "Valid RM OK FOR USE" text is detected!
